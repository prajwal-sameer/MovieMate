# Movie Recommendation System (C)

A C project by Prajwal-Sameer that recommends movies
based on selected genres.

## Genres
- Romance
- Horror
- Sci-Fi
- Comedy
- Action

## Folder Structure
- src → C source code
- docs → algorithm & description
- screenshots → output images

## How to Run
gcc main.c -o movie
./movie
